# 📈 Grafik Çizici (matplotlib) / Graph Plotter (matplotlib)

## 📌 Açıklama

Bu proje, Python'da matplotlib kütüphanesi kullanılarak basit bir grafik çizimi örneğidir. Sinüs dalgası grafiği çizilir.
